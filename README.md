# Description
This website (located at http://www.sleeplesscrew.com) is built using Node.js and deployed using Heroku. I built it as a side project for a UCLA dance organization to help facilitate the events/booking process. Viewports for this website have been configured for viewing on mobile phones and tablet devices.

#Language
Written in HTML, CSS, and JavaScript

#Dependencies
- AngularJS
- Express.js
- Node.js
- NodeMailer
